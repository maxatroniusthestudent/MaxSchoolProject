Контрольні питання

1. Що таке HTML?

HTML (HyperText Markup Language) — це мова розмітки, яка використовується для створення структури веб-сторінок.

2. Які інструменти необхідні для роботи з мовою HTML?

Потрібен текстовий редактор (наприклад: VS Code, Notepad++) та веб-браузер для перегляду результатів.

3. Які теги використовуються для форматування тексту?

Використовуються теги: <b>, <i>, <u>, <h1>–<h6>, <p>.

4. Як створити таблицю?

Використовуються теги: <table>, <tr>, <td>, <th>.

5. Які елементи використовуються для роботи із зображеннями?

Використовується тег <img> з атрибутами src (шлях) і alt (опис).
